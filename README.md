# 💰 CLI Expense Tracker

A simple command-line expense tracker built using Python.  
This project allows users to record daily expenses, categorize them, and calculate total spending directly from the terminal.

---

## 🚀 Features

- Add expenses with amount and category
- View all recorded expenses
- Automatically calculates total expenses
- Persistent storage using text file
- Basic input validation

---

## 🛠️ Technologies Used

- Python 3
- File Handling
- Command Line Interface (CLI)

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/cli-expense-tracker.git
   ```

2. Navigate into the folder:
   ```bash
   cd cli-expense-tracker
   ```

3. Run the program:
   ```bash
   python3 main.py
   ```
